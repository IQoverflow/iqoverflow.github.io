初始化
git init

添加
git add .

提交至本地
git commit -m “blog”

创建
git remote add origin git@github.com:iqoverflow/iqoverflow.github.io.git

登陆github手动创建仓库 （git提交代码需要的SSH那些操作就不废话了）

提交至github
git push -u origin master

输入账号密码即可 （ssh就不需要账号密码了）
